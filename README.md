# HR Attrition Predictor 🔮

## 📊 Projet Data Mining - Prédiction d'Attrition des Employés

### Objectif
Prédire le risque d'attrition des employés (départ volontaire) pour permettre à la direction de mettre en place des actions de rétention.

### Fonctionnalités
- 🔮 **Prédiction** : Estimation du risque d'attrition en temps réel
- 📊 **Visualisation** : Analyse exploratoire des données
- 📈 **Clustering** : Segmentation non supervisée des employés (K-Means)
- 🔗 **Association** : Règles d'association (Apriori)
- 📄 **Rapport** : Synthèse complète du projet

### Méthodologie (CRISP-DM)
- Business Understanding
- Data Understanding & Preparation
- Modeling (6 algorithmes testés)
- Evaluation & Deployment

### Résultats
- **Meilleur modèle** : Régression Logistique
- **Accuracy** : 85.03%
- **MCC** : 0.394
